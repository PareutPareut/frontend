container vs VM  
VM : 한 운영체제 위에서 동일한 어플리케이션 고립된 다른 환경에서 구동하기 위해서 필요

경량화된 컨셉이 container -> 개별적인 container로 고립된 환경에서 구동, 운영체제 포함하지 않고 container engine(도커)이 설치된 host OS를 이용
